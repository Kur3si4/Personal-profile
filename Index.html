HTML as One of My Core Skills

HTML (HyperText Markup Language) is a markup language used to structure content on the web. It defines the meaning and organization of content using elements and tags.

Structural Elements

Structural elements such as html, head, and body define the basic structure of a web document.

Text Formatting Elements

Elements like h1 to h6 are used for headings, p for paragraphs, strong for important text, and em for emphasized text.

Lists

Lists are used to group related items. ul is used for unordered lists, ol for ordered lists, and li for list items.

Links

The a tag is used to create hyperlinks that allow navigation between web pages.

Example: Visit the W3C website

Images

The img tag is used to embed images in a webpage and uses attributes such as src and alt.

Tables

Tables are used to display data in rows and columns using table, tr, th, and td elements.

Tag Purpose
tr Defines a table row
td Defines table data

Forms

Forms are used to collect user input using the form element and input fields.







Semantic Elements

Semantic elements such as header, section, and footer describe the meaning of content and improve accessibility.
